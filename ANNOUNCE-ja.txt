VFlib は C 言語で書かれたフォントライブラリで、指定したフォン
トのビットマップを得るための関数群を持っています。VFlib の特徴は、
さまざまなフォーマットのフォントをフォーマットの違いを気にする
ことなく統一的に使うことができるところにあります。

VFlib では、以下のフォントフォーマットがサポートされています。

    TeX 関連:       PK, GF, VF, TFM
    Omega TeX 関連: OFM (レベル 0),OVF
    X Window 関連:  PCF, BDF
    その他:         TrueType, Type 1, HBF, 書体倶楽部, JG, e漢字

    * TrueType フォントのサポートには、FreeType ライブラリ 
      第 1.2 版を使用します。VFlib のインストール前に、
      FreeType をインストールして下さい。
    * Type 1 フォントのサポートには、T1Lib ライブラリ
      第 5.1 版を使用しています。VFlib のインストール前に、 
      T1Lib をインストールして下さい。

第三版では、日本語漢字に限らず、さまざまな言語のフォントを取り扱うこ
とができます。(第一、二版では、日本語漢字のみの対応でした。)

VFlib を利用した TeX のプレビューアーやプリンタドライバの TeX-Guy 
パッケージ TeX-Guy も開発しています。御利用下さい。

なお、現時点では FreeBSD 5.3 での動作確認しかしていません。
Windows {3.1,95,98,NT} などでインストールするには、Makefile 
の書き換え等が必要となります。
（コンパイルには GNU autoconf や GNU libtool を使っていますので、すこし
めんどくさいかも知れません。）
しかし、ソースコード自体は移植しやすいよう、Unixに依存した部分はほとんど
ありません。移植に成功したら御連絡下さい。
